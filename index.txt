--Ecommerce-WordPress-Like-App

--This App has 2 user type - Buyer
                           - Shop
                           - Viewer

--This App targets small business that wish to have an lively ecommerce app 
  to rum thier business online but have not been a able to so due to some reasons.
  (reasons:)

--Upon creating an account,based on the account type:
      if account_type === Buyer?
          create a normal user role account;
    if account_type === 'shop'?
         ask for some proves,credential ,certs,others to valid if it a valid shop;
         create a Shop(Space) for this account.

--Viewer: - Can only view a particular shop and the products available;
          -  Can view all products;
        
--Buyer - Has all the privilages of a Viewer;
        - Can order from a particular shop;
        - Can  buy from a shop;
        - Can  make a secure online payment(MOMO <All network>)
        - Can Upgrade account to a shop account.

--Shop - Can own a shop -Post available products for the shop
                        -Receive payment on the shop;
                        -Has the privilages of a Buyer;
                        -Can downgrade account to a Buyer;
                        -Receive order from buyers;
